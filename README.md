# CUNDI REFERENCIAS

proyecto web para recopilar y consultar referencias de los profesores de la universidad de cundinamarca

(web proyect to collect and consult references about UdeC´s teachers)

ofrecerles a los estudiantes un pagina donde pueden consultar la reputacion de los profesores y agrgar comentarios acerca de los mismos, con el fin de considerar con cual profesor conviene mas inscribir materias 
(ofert to the UdeC´s students a web page where they can consult the UdeC´s teachers reputation and add comments about them, in order to consider which teacher is more advisible enroll subjets)
